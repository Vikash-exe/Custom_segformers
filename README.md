# Custom_segformers
This work starts with SegFormerB2 and tries to increase its edge detection and overall accuracy.

The final notebook is: Custom_segformers\9_channel_AGU_CA_Pseudo_final.ipynb
Rest of the notebooks are different itirations during development.

Major adaptations are as follows:
    1. Hybrid Loss function
    2. Decoder pipelie changes with AGU
    3. Cross-attention
    4. Pseudo-labeling

Hybrid Loss Function:
    The work required multiple feature penalization, and to cater all different loss function with their algorithm, a hybrid loss containing CE, Lovasz and Tversky is developed.

Decoder:
    Attention Guided Up-sampling is introduced for better edge detection for different class.
    this includes a 1X1 convulation for change in pixels and a learnable weight is aaded.

Cross-attention:
    Since the model needed to have synergy of all the bands togehter, individual self attention heads are attached to each of the channel. During training, these different heads learn which channel is more dominating for different classes.

Pseudo-labelling:
    Since the training is carried out on mask generated by random forest classifier, the mask is 84% accurate with ground truth points. Since the model is predicting with good accuracy, we can provide the pseudo-labels for higher accuracy. The threshold is set to 92% so if the model is confident enough then only it will be used to masking.
